enum grading::TestRunStrategy
strong

Ways that we can run tests.

We can select among these at run-time with the command-line argument --strategy=inline|separated|sandboxed.

Enumerator
Inline 

In the same process, in the current call stack.

Separated 

In separate but unsandboxed processes.

Sandboxed 

In a separate, sandboxed process (if supported).
